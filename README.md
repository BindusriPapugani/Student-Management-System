# Student Management System

A simple Student Management System built using React.js.  
This project performs CRUD operations (Create, Read, Update, Delete) using Axios and JSON Server.

## Features

- Add new student details
- View all students in table format
- Update student information
- Delete student records
- Course selection using dropdown
- Clean UI
- Routing using React Router DOM

## Technologies Used

- React.js
- React Hooks
- React Router DOM
- Axios
- JSON Server
- CSS3

## Concepts Practiced

- useState Hook
- useEffect Hook
- Form Handling
- Controlled Components
- Event Handling
- API Integration
- CRUD Operations
- Routing in React
- Dynamic Routes
- Conditional Rendering

## Project Structure

- Create Student Component
- Read Delete Component
- Update Student Component
- Navigation Bar
- JSON Server Backend
